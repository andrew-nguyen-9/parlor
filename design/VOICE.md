# VOICE

The Secret Order speaks: a haunted, wealthy, well-travelled Victorian house.
Mysterious and intriguing, dry wit — never spooky-cheesy, never nightlife.
Commitment beats fidelity: the mansion voice reaches EVERY surface — empty
states, errors, toasts, 404, loading — not just headings (poolsuite rule).

## Tone
formality 3/5 · jargon 1/5 · humor 2/5 — playful within a serious, antiquarian
frame. Bends: the moment a question or its answer is on screen, drop the theatre —
prompts and results are plain, direct, and unambiguous (legibility floor). Theatre
lives in chrome (nameplates, blurbs, transitions), never in the trivia itself.

## Terminology
| always say | never say | because |
|------------|-----------|---------|
| room | page / screen | every game is a room in the house |
| the deck / a card | menu / tile grid | the home is a deck; each game is one card |
| the Order, the house | we / the team / PARLOR Inc. | the diegetic narrator, not a company |
| lobby | home / back | RoomShell's exit says "Lobby" |
| character title (the Host, the Witch, the Medium…) | generic host / bot | each room has one Secret Order character (canon in GAMES.md) |
| candle / flame / gutters out | timer / time's up | ritual metaphor for time |
| forged / the bank | generated / database | questions are forged nightly from facts |

Room names are proper nouns (Sanctum Mysterii, Codex, Chronos, Fractures, Ignite,
Atlas Obscura, The Gauntlet, Thread of Fate, The Séance, Climb of the Initiate,
The Overture). Category names title-case: History, Music, Sports, Screen, Geography, Wildcard.

## Microcopy patterns
errors: name what happened + the way forward, in-character but clear — "The
archive is unreachable; the house has laid out last night's cards instead." (states
the seed-bank fallback). Never a raw stack trace, never blame the player.
empty states: first-run = an invitation ("The table is set. Choose a card."); cleared
= calm, never a dead end. Always in the mansion voice, always with a next action.
confirmations / CTAs: verb-first, ≤3 words, Title or sentence case, no terminal
punctuation — "Deal the cards", "Drop the pin", "Name the culprit", "Enter the room".
Second person to the player; the Order speaks first person only as narrator.
numbers & scores: plain digits, no theatre inside a score readout (legibility);
flourish lives in the surrounding frame, not the number.
